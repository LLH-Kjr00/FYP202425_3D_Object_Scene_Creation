How to Install the Image Processing Program:
1. Download the ZIP file  
2. Extract the ZIP file 
3. Open the file directory where "imageprocessing_App.py" is
4. Run "pip install -r requirements.txt" to install all the required libraries 
5. Installation is finished, ready for operation 

How to Use the Image Processing Program:
1. Load the images of the object categorized as sub0folders and put the sub-folders in the “Input” folder
2. Run imageprocessing_App.py
3. Type in the folder name under the “Input” folder to choose the images for processing
4. Wait for a moment 
5. Processing is finshed and the images are stored in the “Output” folder under a subfolder with the same inputted folder name
